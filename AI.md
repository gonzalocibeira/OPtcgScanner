# AI Usage Notes

This repository welcomes AI-assisted contributions. The notes below outline expectations when using AI tools while working on this project.

## Allowed uses

- Drafting documentation, README updates, or user guides.
- Suggesting code improvements or refactors.
- Summarizing changes for release notes.

## Data handling

- Do **not** submit private or sensitive information to AI tools.
- Avoid sharing camera feeds, screenshots, or captured card data outside this repository.
- If you must include example data, use fabricated or publicly available samples.

## Verification requirements

- Always run through the script or check usage flags after AI-suggested edits.
- Verify macOS-specific steps and dependencies are accurate.
- Confirm any external API references are current and documented.

## Attribution

If AI assistance materially influenced a change, mention it in the commit message or PR description.
